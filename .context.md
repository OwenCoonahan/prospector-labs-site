## Current Stack

- Pure CSS, no frameworks
- Minimal JS for interactions

### Typography
- Nav: 14-15px
- Headlines: Large, 600 weight, tight tracking
- Body: Gray-500 for secondary text

### Layout
- Fixed nav at top
- Full-width hero sections
- 32px padding on nav
- Clean section dividers

### Aesthetic
- White backgrounds (light mode)
- Black CTAs
- 1px gray borders
- Hover states on links
- No gradients, no shadows

### Components
- Fixed nav with logo + links + CTA
- Hero section with headline
- Project grid (hover reveals)
- Stats row
- Footer CTA
